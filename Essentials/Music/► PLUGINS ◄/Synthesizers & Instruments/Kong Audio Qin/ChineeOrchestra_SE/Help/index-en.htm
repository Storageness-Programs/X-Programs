This is a very quick, rough and short help file showing you some basics of QIN RV engine. New users are encouraged to check out our video tutorials and read the QIN-Powered Instrument User Manual too.

Start the host sequencer of choice and load an instance of QIN RV. Click the "Browser" button, and you will see the Instruments Menu (only installed instruments are shown) and the

 Effects Menu. Click once to select an instrument, then click the "LOAD" button to load it up into QIN RV. Now QIN RV is ready for use.

   
 

 


 

You can click the instrument icon on the left side of the rack to show up and switch between

the user interface of different instruments.

The first loaded instrument is set to MIDI channel 1 by default. If more than one instrument is

 loaded, QIN RV will assign MIDI channels automatically. The default audio output channel is

 always assigned to Out 1. The audio output channel can be manually changed if needed.

START

*Click the preset loading window to display the preset list menu. Select one of the presets to load it. Details are as follows:

I. We will use patch Qudi_Main_Leg as an example.

"Sus" means the patch is made of long sustained notes; "Leg" means it is in legato mode; By looking at this patch name combination, we can see it is a patch made of sustained notes in the legato mode.

Select the "EXTENDERS" tab, then click the "Legato II" icon. Here we have access to the Legato Assignment II Extender. Subset loaders 01/02 are assigned to "Normal"; loader 09 is assigned to "Up", which means that the soundset loaded in loader 09 is triggered by the upward legato playing notes; when there are no "Down" legato assigned, both upward and downward legato playing will trigger load 09. Click the “OK” button to return to the EXTENDERS tab.

II. Now we will use patch Pipa_Main_MW_Leg for further examination.

 

 

 

 

 

 

"Pipa" is the name of the instrument in this patch."Main" means the as a presiding instrument; "Leg" means it is in legato mode; "MW" means some soundsets are triggered by Modwheel; By looking at this patch name combination, we can see it is a patch made of vibrated notes and is played in the legato style, along with Modwheel-triggered expressions.

 

On the LAYERS tab, we can see 12 soundset loaders and their current usage. Click the "Edit" button on the right of each loader will let you access the subset edit page; On the KEYMAPS tab, you can view and change the velocity mappings.

 

On the EXTENDERS tab, click the "Mod Wheel" icon. Here we have access to the Modulation Wheel settings page. We can see that loaders 01/02/09 are assigned to "Normal", while loader 07/08 are assigned to "Low", which means that when the incoming MIDI message is between 64-127, the subset loaded in loader 01/02 will be triggered. Click the “OK” button to return to the EXTENDERS tab.

 

On the "EXTENDERS" tab, click the "Legato II' icon, and we can see the loader 01/02 are set to the "Normal" group; 09 is assigned to "Up", which means loaders 09 will be triggered by upward legato playing notes. Click the “OK” button to return to the EXTENDERS tab.
