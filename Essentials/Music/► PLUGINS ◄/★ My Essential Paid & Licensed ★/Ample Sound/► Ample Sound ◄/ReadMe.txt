*************************************************************
	How To Install The Library
*************************************************************
1. For Windows:
  Double click on "Library Installer.exe"

  The installer will locate the pkg file in the same folder and start installation, and if it is unable to locate the pkg file due to system error, please manually drag the pkg file into the installer window.

2. For Mac:
  Double click on the pkg file in this folder


Please do not use any special character in the library installation path, such as "~!@#$%^&*+={}[]|?<>,':;".

If you have any question with regard to library installation of the products from Ample Sound, please check the information at: https://www.amplesound.net/en/Installation_and_activation.pdf
or you can contact us by email: service@amplesound.net
